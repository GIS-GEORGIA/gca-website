# Georgian Cartographers Association — Website

საქართველოს კარტოგრაფთა ასოციაციის ოფიციალური ვებ-გვერდი.

🌐 **[gca.qgis.ge](https://gca.qgis.ge)**

---

## ტექნოლოგიები

- [Next.js 15](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS
- Leaflet — ინტერაქტიული რუკებისთვის

## გაშვება ლოკალურად

```bash
npm install
npm run dev
```

გახსენი [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
